# Estrutura de Documentação Educacional

Este diretório contém a documentação do projeto Neural Web Tester, com foco em ensinar desenvolvedores a implementarem redes neurais simples.

## Estrutura
- `education/`: Conceitos teóricos sobre IA e Visão Computacional.
- `technical/`: Explicação técnica detalhada do funcionamento do código.
- `tutorials/`: Passo a passo prático para rodar e estender o projeto.
